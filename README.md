## Project Description
This project focuses on studying VLC Media Player as an open-source software. It includes basic analysis of its features, usage in Linux, and understanding how open-source tools work in real systems.

## Shell Scripts

### Script 1: System Identity Report
Displays system information such as kernel version, user, uptime, and OS details.

### Script 2: FOSS Package Inspector
Checks whether VLC is installed and displays its version, maintainer, and description.

### Script 3: Disk and Permission Auditor
Analyzes important system directories for permissions and disk usage.

### Script 4: Log File Analyzer
Analyzes log files to count occurrences of a keyword and displays recent matching lines.

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source statement based on user input and stores it in a file.
